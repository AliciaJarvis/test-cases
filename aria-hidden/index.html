aria-hidden test case

When set to "true", the aria-hidden attribute removes the element from the accessibility tree. When applied to a <span> element inside a <button> element, Firefox does not honour the hidden state of the <span>.

Inside a button

The browser should not include the <span> in the accessibility tree, and screen readers should not announce the "XXX" content of the <span> element.

Outside of a button

The browser should not include the <span> in the accessibility tree, and screen readers should not announce the "XXX" content of the <span> element.
