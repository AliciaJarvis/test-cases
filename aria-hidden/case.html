Case: aria-hidden

When the aria-hidden attribute is used within a <button> element, the hidden state of CSS generated content is ignored in Firefox.

Without <button>

Example code

With <button>

Foo <span aria-hidden="true" class="icon"></span>

Expected results

The content generated with the CSS after selector and hidden with aria-hidden should not be exposed in the browser accessibility tree.
