CSS before/after test cases

The examples on this page contain content generated with the CSS before/after pseudo selectors. The generated content is hidden from assistive technologies using the aria-hidden attribute.

Firefox honours the aria-hidden state when the content is inside a <span> element. When the span is inside a <button>, Firefox does not honour the aria-hidden state and instead exposes the generated content as the accessible name for the button.

<span> examples

Example code


<span class="a" aria-hidden="true"></span>

<span class="b" aria-hidden="true"><span></span></span>

<button> examples

Example code


<button><span class="a" aria-hidden="true"></span></button>

<button><span class="b" aria-hidden="true"><span></span></span></button>
