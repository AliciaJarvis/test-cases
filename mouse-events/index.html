Mouse events test case

The button on this page is a <div> with role="button", with a click event listener.

By default, the Jaws screen reader simulates a mouse click whenever the enter or space keys are used to activate a control. The "Link activation" setting can be set to "send enter key" instead, but this setting does not affect the behaviour of the space key - which will always simulate a mouse click whenever it's used.
